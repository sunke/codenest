## Compile the code
```
$ git clone https://github.com/sunke/codenest/tree/master/design-patterns-ts.git
$ cd design-patterns-ts
$ tsc
```

## Execute the code
Example: execute the demo code for Adapter pattern.
```
$ cd design-patterns-ts
$ node .\build\adapter.js
```
